# Historia del desarrollo de software #
## El desarrollo de software es el proceso de crear programas y aplicaciones que permiten a las computadoras realizar tareas específicas. Su historia está estrechamente ligada a la evolución del hardware y de la informática. ##
**Años 1940–1950: Inicios**
### El software se programaba directamente en lenguaje máquina (ceros y unos). ###

### Los programas eran simples y estaban ligados al hardware. ###

### No existía una separación clara entre hardware y software. ###
**Años 1950–1960: Lenguajes de alto nivel**
### Aparecen lenguajes como FORTRAN, COBOL y ALGOL. ###
### Se facilita la programación y se reduce el tiempo de desarrollo. ###
### Surgen los primeros sistemas operativos. ##
**Años 1960–1970: Crisis del software**
### Los proyectos se volvían más grandes y complejos. ###
### Muchos sistemas fallaban, se entregaban tarde o superaban costos. ###
### Nace la Ingeniería de Software para mejorar calidad y organización. ###
**Conclusión**
### El desarrollo de software ha evolucionado desde programas simples hasta sistemas complejos e inteligentes. Hoy es una disciplina clave en la sociedad digital, presente en casi todos los ámbitos de la vida. ###
